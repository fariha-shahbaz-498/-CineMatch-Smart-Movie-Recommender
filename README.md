# 🎥 CineMatch – Smart Movie Recommender

CineMatch is a modern Movie Recommendation System built using Machine Learning and Streamlit.  
It helps users discover movies based on their interests with a clean and interactive UI.

---

## 🚀 Features

- 🔍 Smart movie search (works even with 1 letter)
- 🎬 Personalized recommendations (Collaborative Filtering)
- 🖼️ Movie posters using TMDB API
- 🌗 Dark mode UI
- ⚡ Fast and responsive interface
- 🎥 Trailer links for each movie
- 🎨 Clean Netflix-style layout

---

## 🧠 Tech Stack

- Python
- Pandas & NumPy
- Machine Learning (Correlation-based recommender)
- Streamlit (Web App)
- TMDB API (Posters & Movie Info)
